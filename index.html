manic sprite dream icons

A sprite is a snazzy composition of multiple images. They are used in web design to reduce the number of server requests a page makes.

Sprites are set as a background image. We use CSS to call out which part of the image we'd like at a certain location on our page by specifying background-position.

The icons on this page were generated using fontawesome as a template. They are being displayed using a compass generated sprite sheet.
